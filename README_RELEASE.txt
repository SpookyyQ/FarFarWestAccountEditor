FarFarWest Unlock all tool

How to use:

1. Keep all files from this folder together.
2. Start `FarFarWest Unlock all tool.exe`.
3. Click `Auto Import Save` to load the default FarFarWest save automatically.
4. Make your changes.
5. Click `Save`.

Important:

- No Python installation is required.
- Keep all included runtime files next to the EXE.
- `Save` creates a timestamped backup before overwriting the current save.
- If auto import finds nothing, use `Open Save Folder` or `Open`.

Default save folder:

`%LOCALAPPDATA%\FarFarWest\Saved\SaveGames`
